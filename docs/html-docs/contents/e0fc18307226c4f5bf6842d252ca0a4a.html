MCMasterDevice

UMLClass
MicroController :: MicroController :: MicroController :: MCMasterDevice

Description

A subclass of MCDevice defining the master controller for the system. Unlike the MCOutputDevice and MCInputDevice classes, MCMasterDevice is a representation of the device MicroController is running on

Generalized Elements

  • MCDevice

Specialized Elements

  • MasterController

Operations

Visibility Name Description
private loop()

Defines the main logic that the Master Controller follows to direct the system. The loop function is automatically run by the MicroControllerMain.cpp file used by MicroController
private build()

Sets up the Controller. Connects to devices, initializes variables, and prepares for main execution
private setup()

Does anything needed to setup the master controller for the next loop. Can be used to simplify the logic in loop(). Is not required for implementation
private teardown()

Cleans up the Master Controller
